# Artafex 4 — Verified SKU Reference

**Source of truth:** DMF A/V Product Guide 2025 V2 (official catalog), Artafex 4 ordering pages.
Every SKU below is **transcribed verbatim** from the catalog and paired with its printed label. OCR spacing (e.g. "M 4 LT S S") has been normalized to the real part number (`M4LTSS`).

**Confidence key:**
- ✅ **Verified** — printed in the official catalog exactly as shown.
- 🟡 **Inferred** — constructed from the catalog's published finish/grammar legend but not printed as a standalone SKU (the catalog prints only the White `WH` variant of each trim). Confirm against a spec sheet before ordering non-white finishes.
- ⛔ **Not in guide** — must be confirmed from the product spec sheet.

## Retrofit (existing-fixture) — Classic Retrofit Trims  ✅

**Source:** Classic Retrofit Trims spec sheet. This is the path for **retrofitting an existing 4", 5", or 6" can** (e.g. an old MR16/PAR fixture) — **no new housing**. The Artafex 4 module drops into the existing aperture via a retrofit trim + adapter.

**Assembly = Module + Retrofit Trim + (optional) Adapter + (if DALI) Interface.**

**Retrofit Trim builder:** `DRD2TR` + aperture(`4`/`5`/`6`) + style(`S` Smooth / `B` Baffle) + finish(`WH` White / `BK` Black). Round only.

| SKU | Description |
|---|---|
| DRD2TR4SWH | 4" Round, Smooth, White |
| DRD2TR5SWH | 5" Round, Smooth, White |
| DRD2TR6SWH | 6" Round, Smooth, White |
| DRD2TR4BWH | 4" Round, Baffle, White |
| DRD2TR5BWH / DRD2TR6BWH | 5"/6" Baffle, White |
| …`BK` | Black variants (e.g. DRD2TR4SBK) |

**Module:** standard Artafex 4 builder — `ART4D…` (Downlight) or `ART4A…` (Adjustable). Same lumens/CCT/beam/dimming/finish grammar as §Modules.

**Adapter (optional):** module ships with a PowerPlug male-to-leads adapter (hardwire). Optional quick-connect adapters:

| SKU | Adapter |
|---|---|
| DRD2X-QCM-ED | Edison (medium screw base) |
| DRD2X-QCM-GU | GU-24 |
| DRD2X-QCM-M | PowerPlug Luminaire Disconnect (Male) |

**DALI interface (if `D` dimming):** `DID2X-<beam>` (Downlight) / `DID4X-<beam>` (Adjustable), e.g. `DID2X-GA`.

> Note: an existing **4" MR16** is low-voltage (GU5.3 + transformer). Retrofit typically means bypassing/removing the transformer and hardwiring (or using the Edison adapter if a medium-base socket is present). The adapter field covers Edison / GU-24 / PowerPlug / hardwire.

## Modules (light engine) — Product Builder  ✅

**Source:** Artafex 4 spec sheets (Round/Square Downlight & Adjustable, standard + PhaseX). The module part number is **built**, not picked from a list, and is **shape-independent** (same module code for Round or Square housings).

**Grammar:** `[ART4D|ART4A]` + `[Lumens]` + `T` + `[CCT]` + `[Beam]` + `[Finish]` + `[Dimming]`

| Segment | Codes |
|---|---|
| Product | `ART4D` Downlight · `ART4A` Adjustable |
| Lumens | `07` 750 lm · `10` 1000 lm · `12` 1250 lm · `15` 1500 lm |
| CRI | `T` True Spectrum® (always present) |
| CCT | `27` 2700K · `30` 3000K · `35` 3500K · `40` 4000K · `3W` Warm Dim (3000–1800K) · `T1` Tunable White (4000–1800K) |
| Beam (Downlight) | `GA` General Ambient 90°+ · `NS` Narrow Spot 15° · `SP` Spot 25° · `FL` Flood 40° · `WF` Wide Flood 60° · `LS` Linear Spread · `SF` Soft Focus |
| Beam (Adjustable) | `NS` · `SP` · `FL` · `LS` Linear Spread 60°×30° · `SF` Soft Focus · `NH` Narrow Spot w/ Hex Louver · `SH` Spot w/ Hex Louver · `FH` Flood w/ Hex Louver |
| Finish (reflector) | *(blank)* Black (default) · `W` White |
| Dimming | `T` TRIAC/ELV · `D` DALI-2 · `X` PhaseX |

**Verified examples (printed on the sheets):**
- `ART4D07T27GAT` — Downlight, 750 lm, 2700K, General Ambient, TRIAC/ELV ✅
- `ART4A07T27NST` — Adjustable, 750 lm, 2700K, Narrow Spot, TRIAC/ELV ✅
- `ART4D10TT1GAX` — Downlight, 1000 lm, Tunable White, General Ambient, PhaseX ✅
- `ART4A07TT1SPX` — Adjustable, 750 lm, Tunable White, Spot, PhaseX ✅

**Module compatibility rules (from spec-sheet footnotes):**
- **Warm Dim (`3W`) and Tunable White (`T1`) → 1000 lm only**, and **not available in Soft Focus**.
- **DALI-2 (`D`)**: not available in Warm Dim/Tunable White except limited (1000 lm tunable/warm, or 1250 lm static CCT).
- **General Ambient (`GA`) and Wide Flood (`WF`) are Downlight-only** (not offered on Adjustable).
- **Hex Louver beams (`NH`/`SH`/`FH`) are Adjustable-only.**
- **Adjustable lumens cap at 1000 lm** (`07`/`10` only).

**Module wattage (verified ✅ — for connected-load calc):**

| Lumens | Watts |
|---|---|
| 750 lm (`07`) | 9.5 W |
| 1000 lm (`10`) | 12.5 W (TRIAC) · **14.5 W** (DALI-2 / PhaseX) |
| 1250 lm (`12`) | 14.3 W |
| 1500 lm (`15`) | 14.3 W |

Adjustable draws the same 9.5 W (750) / 12.5 W (1000). Used by the tool to total connected load per room/job.

**Associated control accessories (verified ✅):**
- PhaseX modules require the **PhaseX Gateway** `ART-GWX` (or `ART-GWX-E` with Ethernet).
- DALI-2 modules use a DALI interface device: `DID2X-…` (Downlight) / `DID4X-…` (Adjustable), suffixed by beam (e.g. `DID2X-GA`, `DID4X-NS`).

### Housing × module compatibility (spec-sheet footnotes)  ✅
- **Standard-rating-only housings** (no Fire-Rated version): Super Shallow (`NS`), Chicago Plenum (`CP`), Double Housing (`DH`), Spray Foam (`SF`).
- **Not available with DALI-2**: Low Profile (`LT`), Concrete (`CC`).
- **Double Housing (`DH`)**: 750 or 1000 lm only.
- **Low Profile (`LT`)** is the TRIAC/ELV 3" plenum housing.

## SKU grammar (corrected)

- **Housing:** `M4` + install(`NC`/`CP`/`DH`/`LT`/`SF`/`RM`/`NS`) + shape(`R`/`S`) + variant(`S` Standard / `F` Fire-Rated). Special: `M4CC` Concrete (round-only, no suffixes).
- **Trim:** `M4T` + shape(`R`/`S`) + style(`S` Standard / `L` Wall-Wash / `P` Pinhole / `H` Hyperbolic / `Q` Round-on-Square) + finish(`WH`/`BK`/`BZ`/`CW`/`CC`) + `FL` if flangeless. **Decorative** = Standard style + finish + decorative suffix (`DCF`/`DCC`/`DCCF`/`DOF`) — e.g. `M4TRSWHDCF`.
- **Accessory:** `M4` + `X`(mud)/`K`(kit)/`C`(collar) + shape(`R`/`S`) + descriptor (`MUD`/`TEMPLATE`/`EXT`/`RETRO6`).
